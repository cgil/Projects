IFProgressBar

The IFProgressBar object creates a new progress bar GUI component. A progress bar is updated with a value bwtween 0 and 1 representing a process's percent complete.

 

Method Summary

IFProgressBar(int x, int y, int width);

x, the X position of the text field's upper left corner
y, the Y position of the text field's upper left corner
width, the width of the text field

setProgress(float progress);

The setProgress() method updates the progress bar's length with a new percent complete value.

progress, the percent complete (between 0 and 1)

getProgress();

The getProgress() method returns a floating point value between 0 and 1, which represents the current percent complete.

setLabel(String newLabel)

sets the progress bar label text

newLabel, the new label for the progress bar

getLabel()

returns a String object representing the progress bar's label text

setWidth(int width)

sets the width of the progress bar

width, the new width for the progress bar

getWidth()

returns the width of the progress bar in integer form.

setHeight(int height)

sets the height of the progress bar

height, the new height for the progress bar

getHeight()

returns the height of the progress bar in integer form.

setSize(int width, int height)

convenience method to set the dimensions of the progress bar

width, the new width for the progress bar
height, the new height for the progress bar

setX(int x)

sets the X position of the progress bar relative to its GUIController

x, the new X position for the progress bar

getX()

returns the integer X position of the progress bar relative to its GUIController.

setY(int y)

sets the Y position of the progress bar relative to its GUIController

y, the new Y position for the progress bar

getY()

returns the integer Y position of the progress bar relative to its GUIController.

setPosition(int x, int y)

convenience method to set the position of the progress bar relative to its GUIController

x, the new X position for the progress bar
y, the new Y position for the progress bar

 

Example

import interfascia.*;

GUIController c;
ProgressBar p;
float percent = 0;

void setup() {
   c = new GUIController(this);
   p = new ProgressBar(10, 10, 80);

   c.add(p);
   p.setProgress(percent);
}

void draw() {
   p.setProgress(percent);
   if (percent < 1) {
      percent += 0.01;
   } else {
      percent = 0;
   }
}
