DOCUMENTATION

IFCheckBox

The IFCheckBox object creates a new check box GUI component.

 

Syntax

IFCheckBox c = new IFCheckBox("Enable", 30, 30);
c.addActionListener(this);

 

Method Summary

IFCheckBox(String label, int x, int y);

label, the text displayed next to the check box
x, the X position of the check box's upper left hand corner
y, the Y position of the check box's upper left hand corner

setLabel(String newLabel)

sets the check box label text

newLabel, the new label for the check box

getLabel()

returns a String object representing the check box's label text

setWidth(int width)

sets the width of the check box

width, the new width for the check box

getWidth()

returns the width of the check box in integer form.

setHeight(int height)

sets the height of the check box

height, the new height for the check box

getHeight()

returns the height of the check box in integer form.

setSize(int width, int height)

convenience method to set the dimensions of the check box

width, the new width for the check box
height, the new height for the check box

setX(int x)

sets the X position of the check box relative to its GUIController

x, the new X position for the check box

getX()

returns the integer X position of the check box relative to its GUIController.

setY(int y)

sets the Y position of the check box relative to its GUIController

y, the new Y position for the check box

getY()

returns the integer Y position of the check box relative to its GUIController.

setPosition(int x, int y)

convenience method to set the position of the check box relative to its GUIController

x, the new X position for the check box
y, the new Y position for the check box

addActionListener(Object listener);

adds an action listener Object which receives GUIEvents when the check box is checked or unchecked. You must implement the actionPerformed() method in order to receive events from the check box.

listener, the object that events are sent to (usually 'this').

isSelected();

returns a boolean value; it is true if the check box is checked, and false otherwise.

 

Example

import interfascia.*;

GUIController c;
IFCheckBox b;

void setup() {
   c = new GUIController(this);
   b = new IFCheckBox("Enable", 20, 20);

   c.add(b);
   b.addActionListener(this);
}

void draw() {
}

void actionPerformed(GUIEvent e) {
	if (e.getSource() == b) {
		if (b.isSelected()) {
			println("Checked");
		} else {
			println("Unchecked");
		}
	}
}
