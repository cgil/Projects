IFLabel

The IFLabel object creates a new text label.

 

Method Summary

IFLabel(String label, int x, int y);
IFLabel(String label, int x, int y, int textSize);

label, the text displayed on the label
x, the X position of the label's upper left hand corner
y, the Y position of the label's upper left hand corner
textSize, text size for the label text

setLabel(String newLabel)

sets the label text

newLabel, the new label text

getLabel()

returns a String object representing the label's text

setWidth(int width)

sets the width of the label

width, the new width for the label

getWidth()

returns the width of the label in integer form.

setHeight(int height)

sets the height of the label

height, the new height for the label

getHeight()

returns the height of the label in integer form.

setSize(int width, int height)

convenience method to set the dimensions of the label

width, the new width for the label
height, the new height for the label

setX(int x)

sets the X position of the label relative to its GUIController

x, the new X position for the label

getX()

returns the integer X position of the label relative to its GUIController.

setY(int y)

sets the Y position of the label relative to its GUIController

y, the new Y position for the label

getY()

returns the integer Y position of the label relative to its GUIController.

setPosition(int x, int y)

convenience method to set the position of the label relative to its GUIController

x, the new X position for the label
y, the new Y position for the label

 

Example

import interfascia.*;

GUIController c;
IFLabel l;

void setup() {
  c = new GUIController (this);
  
  l = new IFLabel ("Descriptive Text", 30, 30);

  c.add (l);
}
