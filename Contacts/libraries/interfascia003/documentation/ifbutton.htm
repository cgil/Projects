DOCUMENTATION

IFButton

The IFButton object creates a new button GUI component.

 

Syntax

IFButton b = new Button("Click", 30, 30);
b.addActionListener(this);

 

Method Summary

IFButton(String label, int x, int y);
IFButton(String label, int x, int y, int width);

label, the text displayed on the button
x, the X position of the button's upper left hand corner
y, the Y position of the button's upper left hand corner
width, the width of the button

setLabel(String newLabel)

sets the button label text

newLabel, the new label for the button

getLabel()

returns a String object representing the button's label text

setWidth(int width)

sets the width of the button

width, the new width for the button

getWidth()

returns the width of the button in integer form.

setHeight(int height)

sets the height of the button

height, the new height for the button

getHeight()

returns the height of the button in integer form.

setSize(int width, int height)

convenience method to set the dimensions of the button

width, the new width for the button
height, the new height for the button

setX(int x)

sets the X position of the button relative to its GUIController

x, the new X position for the button

getX()

returns the integer X position of the button relative to its GUIController.

setY(int y)

sets the Y position of the button relative to its GUIController

y, the new Y position for the button

getY()

returns the integer Y position of the button relative to its GUIController.

setPosition(int x, int y)

convenience method to set the position of the button relative to its GUIController

x, the new X position for the button
y, the new Y position for the button

addActionListener(Object listener);

adds an action listener Object which receives GUIEvents when the button is clicked or released. You must implement the actionPerformed() method in order to receive events from the button.

listener, the object that events are sent to (usually 'this').

 

Example

import interfascia.*;

GUIController c;
IFButton b1, b2;

void setup() {
  c = new GUIController (this);
  
  b1 = new IFButton ("One", 30, 20, 40, 17);
  b2 = new IFButton ("Two", 30, 60, 40, 17);

  b1.addActionListener(this);
  b2.addActionListener(this);

  c.add (b1);
  c.add (b2);
}

void draw() {
}

void actionPerformed (GUIEvent e) {
  if (e.getSource() == b1) {
    println("Button one was clicked");
  } else if (e.getSource() == b2) {
    println("Button two was clicked");
  }
}
